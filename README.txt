This is a King's Midas Miner inspired clone made for test

To run the game please go to:
/Builds/MidasMiner_PC/MidasMiner.exe

Please install "Required_VC_Redist_c86" VC 2013 if necessary

If you build the game from scratch please copy all .dll files and "assets" folder and his subfolders in the same folder of you final .exe file

The project is also hosted on GitHub
https://github.com/HoMeBoYErik/MidasMiner/

=================

Game Rules:

- simply drag or click to swap gems and match a 3 chain or more
- you have 60 seconds to score the more points you can
